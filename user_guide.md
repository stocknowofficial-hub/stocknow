# 📖 Stock Now 사용자 가이드

> 최종 업데이트: 2026-04-06
> ※ 이 서비스의 모든 정보는 참고 목적입니다. 투자 판단과 손익의 책임은 본인에게 있습니다.

---

## 1. 서비스 소개

**Stock Now**는 주요 증권사 리포트, 트럼프 SNS, 글로벌 매크로 지표를 AI가 자동 분석해 예측 신호를 제공하는 프리미엄 서비스입니다.

| 채널 | 내용 |
|------|------|
| 텔레그램 봇 | 실시간 급등주 알림, 시황 브리핑, 트럼프 SNS 분석 |
| 웹 대시보드 | 주간 컨센서스, 예측 성과 기록, 매크로 시그널 |

---

## 2. 시작하기

### Step 1. 웹 가입 (구글 로그인)

1. [stock-now.pages.dev](https://stock-now.pages.dev) 접속
2. **구글 계정으로 로그인** 클릭
3. 구글 계정 선택 후 승인
4. 대시보드 자동 이동 → **14일 무료 체험** 자동 시작

### Step 2. 텔레그램 봇 연동

1. 텔레그램에서 **@Stock_Now_Bot** 검색 또는 링크 접속
2. `/start` 입력
3. 웹 대시보드 → **설정** → **텔레그램 연동** 에서 코드 입력
4. 연동 완료 시 알림 수신 시작

> 텔레그램 연동 없이도 웹 대시보드는 이용 가능합니다.

---

## 3. 텔레그램 알림 종류

### 🚨 실시간 급등주 알림
- 국내/미국 시장에서 **±3% 이상 급등락** 종목 즉시 알림
- 시가총액 100위 이내 필터링으로 신뢰도 높은 종목만 제공
- AI가 급등 이유 분석 포함

### 📊 시황 브리핑
- 한국장: 오전 8:50 / 점심 12:40 / 장마감 16:30
- 미국장: 밤 22:50 / 새벽 2:40 / 5:30
- 시장 분위기, 주요 뉴스, 주목 종목 포함

### 🏛 트럼프 SNS 분석
- 트럼프 Truth Social 게시글 2분마다 모니터링
- 시장 영향 있는 글 감지 시 즉시 AI 분석 → 알림 발송

### 🐳 고래 현황판
- 국내/미국 거래량 & 수급 현황 실시간 업데이트
- Telegraph 링크 형태로 제공

## 4. 웹 대시보드 주요 페이지

### 🧭 주간 컨센서스 (`/consensus`)

이번 주 증권사 리포트를 종합한 시장 방향 분석 페이지입니다.

| 섹션 | 내용 |
|------|------|
| AI 통합 분석 | 매크로 + 리포트 + 트럼프 SNS 종합 주간 시그널 |
| 매크로 시그널 | CNN Fear&Greed 지수 + CBOE VIX 게이지 |
| 이번 주 컨센서스 | 증권사별 상승/하락 방향 집계 |
| TOP 10 시그널 | 신뢰도 높은 예측 상위 10개 (필터 기능 포함) |
| 증권사 리포트 상세 | 각 리포트 근거, 액션 추천, 관련 종목 아코디언 |

**필터 사용법** (TOP 10 시그널):
- `전체 / 📈 매수 / 📉 매도` — 방향 필터
- `전체 / HIGH / MED / LOW` — 신뢰도 필터
- `전체 / ⏰ 7일 이내 / 14일 이내` — D-day 필터
- 각 항목 클릭 → 아래 상세 리포트로 자동 스크롤

### 🎯 예측 성과 기록 (`/history`)

AI 예측의 실제 적중 여부를 추적하는 페이지입니다.

| 섹션 | 내용 |
|------|------|
| 전체 누적 통계 | 전체 기간 적중률 + 평균 수익 |
| 이번 달 통계 | 당월 전체 예측 대비 적중률 |
| 적중한 예측 | 최근 2개월 · 최고 수익률 순 정렬 |
| 진행중인 예측 | HIGH 신뢰도 → 방향 일치 중 → 만료 임박 순 |
| 빗나간 예측 | 만료 후 미달성 예측 목록 |

**카드에서 확인할 수 있는 정보:**
- 진입가 → 현재가 (±%)
- 최고 수익률 달성일
- 월가/네이버 목표가
- AI 액션 추천 (매수 고려, 관망 등)
- 손절 기준

### 🏛 트럼프 마켓 임팩트 (`/trump`)

트럼프 Truth Social 게시글이 시장에 미치는 영향 분석 페이지입니다.

## 5. 구독 플랜

| 플랜 | 기간 | 혜택 |
|------|------|------|
| 무료 체험 | 가입 후 14일 | 전체 기능 이용 |
| 1개월권 | 33일 | 전체 기능 이용 |
| 6개월권 | 186일 (3+3) | 전체 기능 이용 |
| 1년권 | 368일 | 전체 기능 이용 |

### 구독 방법
운영자에게 문의하면 텔레그램 구독 링크를 받을 수 있습니다.

### 친구 추천 보너스
- 내 추천 링크로 친구가 가입하면 **+14일 연장**
- 최대 60일까지 누적 가능
- 추천 링크: 웹 대시보드 → **설정** → **추천 링크 복사**

## 6. 자주 묻는 질문

**Q. 텔레그램 알림이 안 와요.**
→ 봇과의 대화가 차단되어 있지 않은지 확인하세요. `/start` 를 다시 입력해보세요.

**Q. 웹 페이지가 로딩이 안 돼요.**
→ 구글 로그인이 되어 있는지 확인하세요. 시크릿 모드에서는 로그인이 유지되지 않습니다.

**Q. 예측이 적중/빗나감으로 언제 표시되나요?**
→ 매일 KST 15:40 (국내), 06:10 (미국) 에 자동 판정됩니다. 예측 만료일까지 달성 못 하면 빗나감 처리됩니다.

**Q. 무료 체험 종료 후에는 어떻게 되나요?**
→ 구독 만료 후에는 프리미엄 페이지 접근이 제한됩니다. 구독 갱신 후 이용 가능합니다.
